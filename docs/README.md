# Pahal Retail – Static Marketing Page

Single-page static landing site for **Pahal Retail** (Pahal Technologies).

## Run locally

Open `marketing-site/index.html` in a browser, or run a simple local server.

### PowerShell (no install)

```powershell
cd "D:\Web Development\Billing_App_UI\marketing-site"
python -m http.server 5173
```

Then open `http://localhost:5173`.
