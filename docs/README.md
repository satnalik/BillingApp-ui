# Pahal Retail Static Marketing Page

Single-page static landing site for **Pahal Retail** by Pahal Technologies.

## Run locally

Open `docs/index.html` in a browser, or run a simple local server from this
folder.

### PowerShell

```powershell
cd "D:\Web Development\Billing_App_UI\pahal-billing-web\docs"
python -m http.server 5173
```

Then open `http://localhost:5173`.

## Current positioning

The page now reflects the app's current modules:

- Billing, bill history, and dues
- Product and supplier masters
- Supplier-linked product mapping
- Purchase / stock inward bills
- Purchase cancellation with stock reversal and audit trail
- Reports and admin settings
